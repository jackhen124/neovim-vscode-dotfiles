## Tab/File

- move tab (left / right) within same window
  - ctrl shift (pg up/pg dn)
- move tab (left/right) between different windows
  - ctrl alt (←/→)
- navigate to (left/right) tab
  - ctrl (pg up/pg dn)
- close file
  - ctrl shift w q

## VSCode Text Editing

- sort selected lines alphabetical
  - ctrl alt f
- copy paste from system prefix (Vim-normal)
  - "+
- removed unused imports
  - alt shift o
- Fix autofixable problems eslint (custom)
  - alt shift m
