# Neovim VsCode Dotfiles

This project s to customize my development experience in VS Code.

This is intended to be used with the [VSCode Neovim](vscode:extension/asvetliakov.vscode-neovim) extension

# Dependencies

- [neovim](https://neovim.io/)
- [python](https://www.python.org/downloads/)

# Running Scripts

Copy the configs from their normal place the machine filesystem, and write them to this repo

```bash
python3 copy-machine.py
```

Copy the configs from this repo and write them to the machine to be used

```bash
python3 overwrite-machine.py
```
